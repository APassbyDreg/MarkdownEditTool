## How to use

just run it XD


```
java -jar [file location]/test.jar
```

## Functions

- available MD syntax:
    - lists
    - headings (1-6)
    - hyperlink and image
    - bold and italic
    - code blocks
    - horizontal separate line
    - quotes
- output to TXT/HTML file

## Known issues
