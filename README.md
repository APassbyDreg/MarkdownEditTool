## Project Info

**Creator：** Martin_Z_He

**Download：** [latest release](https://raw.githubusercontent.com/APassbyDreg/MarkdownEditTool/master/out/artifacts/test/test.jar)

## Functions

- available MD syntax:
    - lists
    - headings (1-6)
    - hyperlink and image
    - bold and italic
    - code blocks
    - horizontal separate line
    - quotes
- output to TXT/HTML file
